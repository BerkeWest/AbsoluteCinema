Welcome to the Absolute Cinema App.

This is a movie app that uses The Movie Database Api. 

How to run:
1) Download the file.
2) Create a file named "secret.properties" inside the Gradle folder.
3) Put the following code in the file and fill the brackets with your own information.

API_KEY = [Put your Api Key from TMDB here]
BASE_URL = https://api.themoviedb.org/3/
IMAGE_URL = https://image.tmdb.org/t/p/original
AVATAR_URL = https://secure.gravatar.com/avatar
